# 语言安全感

*发表于 2020年1月9日*

发明了一个词叫语言安全感。人在讲自己熟悉的话题的时候会滔滔不绝，越讲越来劲，但一旦涉及到不熟悉的话题就会史诗级词穷，甚至想转移话题。这是因为当人处在熟悉的话题会给人安全感，可以称这个范围为语言安全圈，而多巴胺是与安全感相联系的，也就是说越熟悉一个领域，在陈述相关知识的时候越不会紧张。《你的生存本能正在杀死你》这本书上提到多巴胺其实具有自我保护作用，在自然中兔子遇到狼多巴胺会降低，安全感降低，产生恐惧的情绪，从而促使了兔子逃跑的求生欲，这是一种动物本能。同样在人类身上也是一样，当人类面对未知的东西，安全感无法维持在正常水平，就会产生逃避的情绪，比如像是第一次在台上演讲时产生的紧张感就是源自于多巴胺变化产生的生物本能，这也是为什么经验比理论要重要的原因，积累了越多经验就越不会面对未知情况。同样在语言沟通上也一样，像是雅思口语，如果和朋友一样边走边用英语聊天反而比面对面提问更能表达出想法，因为朋友是熟悉的人，更容易感觉到语言安全感，而面对陌生人时，特别是考官，短时间内多巴胺调整不到正常水平的话，就直接宕机了。
