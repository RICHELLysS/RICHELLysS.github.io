# 注意力集中法和记忆方法

*发表于 2021年3月8日*

有过这样的经历，听课听讲听到一半就犯困，或者走神，如何避免这种情况？我通常会用这样的方法：

尝试注意讲话人说的数字并记住顺序，然后根据数字直觉从脑子里挖取信息。比如说在听一节统计相关的课程，一节课中提到10%，50%，7，4这几个数字相关，那么回想的时候就问自己：什么东西和10%有关，和50%有关，和7有关，和4有关。一开始会很难回想起来，这个时候就需要在笔记本上记下来数字顺序，然后赋予类比意义然后在脑子里进行抽象和发散。

举个例子，数字5发散：五根手指，五芒星，五边形，计算器上的五，键盘上的五，五十而知天命，八卦中五，百分之五，第五人格，课堂上出现的第五个数字，刺客五六七，钩子...（走神了）

如果在走神之前在脑子里回想起课上关于特定数字的知识点，那么链接就搭建成功了，自然答得上来题，失败就算了，丢分就是丢分，看玄学（？）。这个方法适用于短期记忆。

鉴于我非常迷信数字玄学，会赋予不同数字不同的意义以增强印象，通常会在脑子里用不同数字组合来占卜。比如早上醒来看手机，时间是7：34；7是个强大且很受欢迎的数字（仅个人定义），34相加就是7，很巧合，34是17的两倍，17是我喜欢的组合，3和4在奇门遁甲里的定义相位是“木属生长”，所以我判定今天的状态适合快乐学习并且脑子会很好使。在某个地方看到数字我也会不由得记下来，这样的弊端是使得我对某些并不重要的事情反而记得很清楚，很容易走神，不过相对来说，对于锻炼脑子还是很有帮助。

此习惯会促使人稍微变得有些数字强迫症，有个现成的例子就是在排版三亚水族馆里拍的水生物照片时，出现的元素会按一定数字顺序排列：两只鸭鸭，一只兔，一只羊驼，一只鹦鹉，两只鹦鹉（首尾相连）；两只鱼，三只海星，四只水母，许多只像蚯蚓一样的水生物，无数只水生物（递增关系），等等强迫数字元素按一定顺序排列，宛如刚上幼儿园学会数一个苹果，两个苹果的孩童，不过让我觉得特别有趣，因为这赋予了数字生命。或者更有趣的情况是，玩数独，有时会在脑子里自言自语“4你又跑哪儿去了，8不要闪现，1出来插个旗”，久而久之，因为数字的存在变得没有那么孤独，而世界也拥有了魔法，所以注意力和记忆力好的时候，完全就是得益于给数字赋予意义这种奇怪的记忆法。
