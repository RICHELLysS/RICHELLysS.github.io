# Defining Game Rules

*Published on June 8, 2021*

The mode is simple: roll dice + walk the map.

The six patterns on the dice are defined as shown in Figure 1: learning tools, move forward, establish relationships, intersection and conflict, converter, and problem. The patterns on the map are the same shapes as on the dice, except the lines are cut into small grids (the number of grids per map should be the same, though I didn't pay much attention when drawing).

Gameplay: four players per game. Each player starts with 0 drive value and 4 influence value. Each dice roll reduces influence value by 1 and increases drive value by 1. The pattern rolled occupies a grid according to a specific mechanism (this mechanism still needs thinking). When all grids are occupied, the player with the most occupied grids wins.
