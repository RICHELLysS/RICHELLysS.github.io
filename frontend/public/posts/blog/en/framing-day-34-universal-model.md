# Universal Model

*Published on February 25, 2021*

I feel that language cannot well describe the model in my mind, so I draw a diagram to describe my imaginative learning method.

With this diagram, one can well understand the saying "there is more than one way to solve a problem."

Assume the outward radiating lines are specific fields or disciplines, totaling four dimensions. We know everything emerges from chaos, meaning the lines all start from the center and radiate outward. I personally believe all disciplines can be classified into eight categories (this classification is based on the color of the discipline in my mind). The outward radiating rings represent connections between disciplines. Every discipline has direct or indirect connections with others. These connections can be any kind—not necessarily academically fixed, as long as they feel reasonable to you.

With the above premise, any point in this model is a problem. Each line can be rotated as a "tool" to solve problems. The power for rotation is "imagination." You can pick "tools" from any discipline to solve problems. The focus is not on standardization but on whether the problem can be solved. As long as the "tools" can be assembled into a line to achieve problem-solving, that's sufficient. This is cross-disciplinary thinking for problem-solving.

I have always used this model to learn any knowledge. The drawback is that if a certain discipline's dimensions are not high enough, the tools from that discipline cannot be applied.
