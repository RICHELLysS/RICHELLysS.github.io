## Project Overview

- Built a privacy-focused AI web app that provides hairstyle suggestions using React and Tailwind CSS. Integrated Chrome's Built-in AI API for image analysis and translation, with all processing occurring client-side using local browser storage to ensure no user data upload.
- Designed an intuitive UI for photo upload and result display, prioritizing user privacy and real-time interactive feedback.

## Key Features

- **Technical skills**：React, Tailwind CSS, Chrome AI APIs, Client-Side Storage, Privacy-by-Design Development
- **Soft skills**：Focus on User Privacy, Creative Application of Emerging APIs, UI/UX Sensibility
