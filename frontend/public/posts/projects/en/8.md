## Project Overview

- Created an engaging desktop pet application using Kiro's vibe coding (AI-assisted) environment. Implemented a gamified task-reward system where the pet grows based on user completion of real-world tasks, and a “gacha” (blind box) system for unlocking new pet features, driving user engagement.
- Focused on creating a fun, interactive user experience with core game loops within a constrained development timeframe.
